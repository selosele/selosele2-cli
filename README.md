# selosele2-cli

[2022 개인 블로그](https://github.com/selosele/selosele2-backend)의 CLI 클라이언트

## 작업 기간

- 2023.12.29. ~ 2024.02.05.

## 개발 환경

- ```Python``` - v3.10.12
- ```Poetry``` - v1.7.1
- ```WSL2``` - Ubuntu 22.04 LTS

## 개발 환경 구동

1. ```poetry shell```
2. ```poetry run start```

## 실행파일 생성 명령

`pyinstaller -F selosele2_cli/main.py`
